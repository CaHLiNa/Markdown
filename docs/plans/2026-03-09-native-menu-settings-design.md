# Native Menu And Settings Redesign

**日期：** 2026-03-09  
**状态：** 已确认，待实施

## 目标

把 macOS 菜单栏和设置窗口整理成更像原生应用的结构：

- 菜单栏按文档、编辑、插入、格式、视图这几个稳定心智模型重排。
- 去掉重复入口和命名不一致的项，让已实现功能都能在合适的位置被找到。
- 设置窗口从单页 `Form` 升级成原生偏好设置风格，左侧导航、右侧分组内容，视觉更现代，层级更清晰。

## 当前问题

### 菜单栏

- 顶层菜单过多，`外观`、`主题`、`视图` 彼此边界不清。
- 搜索、命令面板、面板切换等“工作流操作”分散在不同菜单里。
- `段落` 菜单实际上承担的是“插入结构”，名称不自然。
- 部分 Markdown 富文本功能已经做了，但菜单分类仍沿用早期阶段的临时组织方式。

### 设置窗口

- 现在是单页 `Form`，信息密度高，但缺少导航、分组节奏和主次关系。
- 字体、版式、输入、导出等配置都堆在一页，扩展性差。
- 视觉上接近“调试页”，不像一个完成度较高的 macOS 偏好设置窗口。

## 设计原则

1. 优先遵循原生 macOS 菜单语义，不把一切都塞进自定义分组。
2. 高频操作放菜单栏，低频或持续性偏好放设置窗口。
3. 一个功能只保留一个主入口，避免菜单栏和设置页之间重复表达同一个控制项。
4. 设置界面保持克制，不做网页后台风，不做过度卡片化。

## 菜单栏信息架构

### 文件

负责文档生命周期和文件 IO：

- 新建标签页
- 打开文件…
- 打开文件夹…
- 最近打开
- 快速打开…
- 关闭标签页
- 保存
- 另存为…
- 导出为
  - HTML…
  - PDF…
- 打印

说明：

- `导出` 统一改成 `导出为` 子菜单。
- `快速打开` 仍保留在文件流里，因为它本质是文档切换。

### 编辑

负责查找、替换和命令触发：

- 查找…
- 替换…
- 查找下一个
- 查找上一个
- 在文件夹中搜索
- 命令面板

说明：

- 不再把搜索面板切换放进 `视图`。
- 这里保留“工作流辅助”而非文本格式。

### 插入

承接原 `段落` 菜单，语义改成“插入结构”：

- 段落
- 标题 1-6
- 表格
- 代码块
- 引用块
- 数学块
- 有序列表
- 无序列表
- 任务列表

说明：

- 用户心智会更接近 Markdown 编辑器，而不是文字处理器。
- 若后续补充分隔线、Front Matter，可继续归到这里。

### 格式

只保留文本层的样式操作：

- 粗体
- 斜体
- 删除线
- 行内代码
- 行内公式
- 超链接
- 图片
- 清除格式

说明：

- `下划线` 与 `高亮` 若底层兼容性或 Markdown 语义不稳定，不作为一级重点。
- 如果保留，也应放在“扩展格式”分组，而不是和核心 Markdown 操作同级。

### 视图

负责界面可见性和编辑状态：

- 切换源码模式
- 专注模式
- 打字机模式
- 显示侧边栏
- 显示标签栏
- 显示文件面板
- 显示搜索面板
- 显示目录面板
- 外观
  - 跟随系统
  - 浅色
  - 深色

说明：

- `主题` 不再单独占一个顶层菜单。
- 编辑器主题和导出主题属于偏好，不适合作为常驻菜单顶层项。

## 设置窗口信息架构

设置窗口改成两栏布局。

### 左侧导航

- 通用
- 编辑器
- 输入
- 导出

### 通用

- 界面模式
- 默认显示标签栏
- 默认开启专注模式
- 默认开启打字机模式

说明：

- 这一组处理应用层的全局偏好，而不是单纯排版参数。

### 编辑器

- 编辑器主题
- 正文字体
- 正文字号
- 行高
- 页面宽度
- 代码字体
- 代码字号

说明：

- 这组会承担视觉预览的主要责任。
- 右侧内容中加入一个轻量预览卡片，动态反映字号、行高、宽度与字体选择。

### 输入

- 隐藏快速插入提示
- 括号自动配对
- Markdown 语法自动配对
- 引号自动配对

说明：

- 这一组专注“输入行为”，不再和外观项混排。

### 导出

- HTML 导出主题
- 后续可扩展 PDF/打印相关偏好

说明：

- 当前先收拢已有导出主题项，保留未来扩展空间。

## 设置窗口视觉方案

- 窗口宽度比当前更宽，右侧内容留足呼吸感。
- 左侧导航使用原生 `List`/sidebar 风格，不做网页式大标签。
- 右侧每组设置采用轻量分组卡片，但边框和阴影极淡。
- 标题、说明文、控件对齐方式参考原生系统设置，不用网页后台布局语言。
- 预览区域使用真实排版样例，而不是纯文字描述。

## 范围边界

本次不做：

- 完整的设置页搜索
- 独立快捷键编辑器
- 多窗口偏好同步 UI
- 帮助中心、更新检查、账户类功能

## 验收标准

1. 菜单栏分类从 7 组压缩成更稳定的原生结构，重复项消失。
2. 已实现的主要编辑功能都能在更合理的菜单中找到。
3. 设置窗口不再是单页表单，而是具备清晰导航和内容分区的偏好设置界面。
4. 视觉风格明显比当前更完整，但仍保持原生、克制。
